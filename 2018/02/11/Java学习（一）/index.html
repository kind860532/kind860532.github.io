Java知识点

Java语言的三种技术架构

  • JavaSE（Java 2 Platform Standard Edition - 旧称J2SE）-标准版

    |-是为开发普通桌面和商务应用程序提供的解决方案，该版本的技术是Java其他两版本的基础，可以完成一些桌面应用程序的开发。

  • JavaEE（Java 2 Platform Enterprise Edition - 旧称J2EE）-企业版

    |-是为开发企业环境下的应用程序提供的解决方案，该技术体系中包含的技术，主要针对于web应用程序开发。

  • JavaME（Java 2 Platform Micro Edition - 旧称J2ME）-小型版

    |-是为开发电子消费产品和嵌入式设备提供的解决方案，该技术体系主要应用于小型电子消费类产品，如：手机中的应用程序等。

    |-注：JavaME不再使用的是因为现在智能手机的系统软件Android和iOS的应用程序开发，均有对应的编程语言。Android虽用到Java语言，但无需专门使用JavaME版本去应对整个开发过程，所以JavaME也就不再被使用。

Java语言的特点：跨平台性

  • 跨平台性

    |-定义：通过Java语言编写的应用程序在不同的系统平台上都可以运行。

    |-原理：在不同的系统平台上安装不同系统平台的Java虚拟机（Java Virtual Machine - 简称JVM），由JVM负责Java程序在系统平台上的运行，实现跨平台性。

    |-JVM：用于解释并执行Java程序的一个应用软件。

  • 问题

    |-问题一：Java语言的跨平台性依赖于JVM虚拟机，那么其他语言也提供类似的虚拟机是不是也就实现了跨平台的特性？那么为什么Java要强调其跨平台的特性呢？
